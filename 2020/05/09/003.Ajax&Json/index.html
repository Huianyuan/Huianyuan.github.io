Aja&Json

Ajax

  1. 概念：asynchronous JavaScript and XML，异步的JavaScript和XML。是一种无需重新加载整个网页的情况下，能够更新部分网页的技术。提升用户体验。

  2. 实现方式：

    1. 原生的JS实现方法

    2. Jqeury实现方式

      1. $.ajax()

        • 语法：$.ajax({键值对})；

      2. $.get()：发送GET请求

      3. $.post()：发送POST请求

Json

  1. 概念：JavaScript Object Notation，JavaScript对象表示法

    • java：

    • JavaScript :

    Json现在多用于存储和交换文本信息。类似XML。但比XML更小、更快、更易解析。

  2. 语法：

    1. 基本规则

      • 数据在名称/值对中：Json数据由键值对构成

        • 值取得类型：
          • 数字（整数或浮点数）
          • 字符串（在双引号中）
          • 逻辑值（true 或 false）
          • 数组（在中括号中） {“persons”:[{},{}]}
          • 对象（在大括号中） {“address”: {“province”: “陕西”…}}
          • null

      • 数据由逗号分隔：多个键值对用逗号隔开

      • 大括号保存对象：使用{}定义Json格式

      • 中括号保存数组：[]

    2. 获取数据

      1. Json对象.键名

      2. Json对象[“键名”]

      3. 数组对象[索引]

      4. 遍历

  3. Json数据和Java对象的相互转换

    • Json解析器：
      • 常见的解析器：Jsonlib，Gson，fastjson，jackson（springMVC内置）

    1. Json转为Java对象

      1. 导入Jackson相关jar包

      2. 创建Jsckson核心对象 OBjectMapper

      3. 调用 ObjectMapper的相关方法进行转换

        1. readValue(Json字符串数据, Class)

    2. Java对象转换为Json

      1. 使用步骤：

        1. 导入Jackson相关jar包

        2. 创建Jsckson核心对象 OBjectMapper

        3. 调用 ObjectMapper的相关方法进行转换

          1. 转换方法：

            • writeValue(参数1，obj):

              参数1：

              ​ File：将obj对象转换为Json字符串，并保存到指定的文件中

              ​ Writer：将obj对象转换为Json字符串，并将Json数据填充到字符输出流中

              ​ OutputStream：将obj对象转换为Json字符串，并将Json数据填充到字节流输出 流中

            • writeValueAsString(obj)：将对象转为Json字符串

          2. 注解：

            1. @JsonIgnore：排除（忽略）属性。
            2. @JsonFormat：属性值的格式化。

          3. 复杂Java对象的转换

            1. List
            2. Map
