conda常用命令

深度学习中创建多个虚拟环境，以防各版本依赖之间的冲突。

这里记录下常用的 conda 命令，还有如何在当前虚拟环境中使用 jupyter 。

查看环境

查看所有已创建的环境： conda env list
查看在某一环境下所安装的包：conda list -n myenv
查看当前环境下的所有第三方库：conda list

创建环境

conda create -n env_name package_names

进入环境

activate my_env 进入一个已创建的环境

离开环境

deactivate

如需要更新包

conda upgrade --all or conda update package_name

如需要安装包

conda install package_name or conda install pandas numpy

卸载包

conda remove package_names

在当前虚拟环境中使用jupyter notebook

切换到你需要使用 Jupyter notebook 的虚拟环境；
然后输入 conda install ipykernel 安装ipykernel库；
之后输入 python -m ipykernel install --user --name <当前环境名> --display-name <当前环境名> 。

完毕，之后在当前环境输入 jupyter notebook 或者在vscode中创建 .ipynb 文件，都可以在本虚拟环境中使用 jupyter。

删除当前的kernel

1 查看安装了哪些虚拟环境kernel（在base或虚拟环境下运行都可以）：
jupyter kernelspec list
2 删除指定的kernel：
jupyter kernelspec uninstall <需要卸载环境名>
