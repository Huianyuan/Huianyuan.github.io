34. Find First and Last Position of Element in Sorted Array（Medium）

题目描述

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 O(log n) 的算法。

提示:

  • 1 <= nums.length <= 104
  • -104 <= nums[i] <= 104
  • nums 为 无重复元素 的 升序 排列数组
  • -104 <= target <= 104

思路

暴力解法的时间复杂度是$O(n)$，尝试使用二分法查找。分别找区间的第一个和最后一个位置，即左边界和有边界。使用闭区间，while(left<=right)这里的right是有意义的值。findLeft查找左边界，只有当 (mid==0||nums[mid-1]<target) （即mid为数组第一个，或者mid-1对应的值小于目标值）和 nums[mid]==target （mid对应值为目标值）同时满足时，当前的mid即为左边界，否则返回-1 ； findRight查找右边界，只有当(mid==nums.length-1||nums[mid+1]>target)（mid为数组最后一个，或者mid+1对应的值>目标值）和 nums[mid]==target（mid对应值为目标值）同时满足时，当前的mid即为右边界。其他情况下，都是寻找区间，更新区间左右边界值。
